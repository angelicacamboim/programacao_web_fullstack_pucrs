Agrupamento de seletores

Agrupar seletores CSS permite que você aplique o mesmo conjunto de regras a vários elementos HTML sem repetir as regras para cada seletor individualmente. Isso torna seu código CSS mais conciso e fácil de manter. Para agrupar seletores, você pode simplesmente listar os seletores separados por vírgulas (,).
