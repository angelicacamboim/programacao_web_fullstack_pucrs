Box Model

No modelo de caixa, os elementos HTML são compostos pelos seguintes componentes:

  • Conteúdo (Content): É a área onde o conteúdo real do elemento, como texto, imagens ou outros elementos, é exibido.
  • Preenchimento (Padding): É a área entre o conteúdo e a borda do elemento. Você pode ajustar o preenchimento usando a propriedade padding.
  • Borda (Border): É a linha que circunda o elemento, separando-o do preenchimento e das margens. Você pode estilizar a borda usando a propriedade border.
  • Margem (Margin): É a área fora da borda do elemento, que separa esse elemento de outros elementos vizinhos. Você pode controlar as margens usando a propriedade margin.
