Box Sizing e Box Shadow

A propriedade CSS box-sizing é usada para controlar o modelo de caixa (box model) de um elemento HTML, especificando como o tamanho total de um elemento é calculado, levando em consideração ou não as bordas e preenchimentos. O valor mais comum para essa propriedade é content-box, que é o valor padrão, e faz com que o tamanho total do elemento seja calculado apenas com base no conteúdo, ignorando bordas e preenchimentos. No entanto, ao definir box-sizing: border-box, você está alterando o modelo de caixa para que o tamanho total do elemento inclua tanto o conteúdo quanto as bordas e preenchimentos.
